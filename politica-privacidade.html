Política de Privacidade

Data de disponibilização: 31/03/2026

A **BJ Soft** tem um compromisso sério com a privacidade e a proteção dos dados pessoais das pessoas impactadas pelas suas atividades. Por isso, criamos este Aviso de Privacidade (o “Aviso”), para informar você como coletamos, utilizamos, processamos e compartilhamos, quando estritamente necessário, dados pessoais quando você utiliza nossos serviços de consultoria e suporte em ERP.

1. CONCEITOS IMPORTANTES

  • **Controlador**: O responsável pelas decisões referentes ao tratamento de dados dos titulares afetados por uma atividade.
  • **Dados Pessoais**: Informações que identificam uma pessoa natural, como por exemplo: e-mail, CPF, telefone e endereço.
  • **LGPD**: Lei Geral de Proteção de Dados Pessoais (Lei n° 13.709/2018).

2. PAPÉIS E RESPONSABILIDADES

A **BJ Soft** é a responsável pelo tratamento dos dados pessoais dos Usuários que interagem com nosso site e landing pages, agindo como Controladora desses dados. Para fins de suporte técnico e consultoria, poderemos atuar como Operadores de dados sob as orientações do Cliente.

3. QUAIS DADOS COLETAMOS?

Coletamos o mínimo possível de Dados Pessoais na prestação dos nossos serviços:

  • **Para Atendimento Comercial**: Nome, E-mail e Telefone via formulários.
  • **Para Suporte Técnico**: Dados necessários para acesso e resolução de incidentes no ambiente do Cliente.
  • **Cookies**: Dados de navegação para melhorar a experiência no site.

4. COMO PROTEGEMOS SEUS DADOS

Adotamos procedimentos físicos, eletrônicos e administrativos que garantem a privacidade e segurança dos seus dados, com medidas técnicas compatíveis com os padrões esperados e boas práticas de segurança da informação. Note que o suporte técnico especializado da **BJ Soft** é pautado pela ética e sigilo profissional.

5. SEUS DIREITOS

Como Titular, você possui os direitos garantidos pela LGPD, incluindo:

  • Confirmação e acesso aos seus dados.
  • Correção de dados incompletos ou inexatos.
  • Eliminação dos dados pessoais tratados com o seu consentimento.

6. CANAL DE COMUNICAÇÃO

Para exercer seus direitos ou tirar dúvidas sobre este aviso, entre em contato conosco através do e-mail: **suporte@bjsoft.com.br**.
